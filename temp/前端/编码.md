# ASCII

>字符集
>
>美国制定的字符编码，将英语字符与二进制对应
>00000000~11111111，总共可以对应256个符号



# Unicode

> 字符集
>
> ASCII编码不足以支持其他国家的符号，不同语言产生不同编码
>
> 编码不统一，同一个二进制数可能被解释成不同符号，引起混乱
>
> 出现了统一的标准Unicode
>
> 0000000000000000~1111111111111111，总共对应2^16个符号
>
> 考虑到不够表示的问题，又增加了一组字符编码，2个16位

- Unicode字符集为每个字符分配一个码位，根据不同编码规则，将码位编码为不同字节

## utf-8

> 编码规则
>
> 以 8 位为一个编码单位的可变长编码
>
> 将一个码位编码为 1 到 4 个字节
>
> 英文单词：1个字节；中文汉字：3个字节；
>


规则介绍：

> 1. 单字节的字符，字节的第一位设为0，对于英语文本，UTF-8码只占用一个字节，和ASCII码完全相同；
>
> 2. n个字节的字符(n>1)，第一个字节的前n位设为1，第n+1位设为0，后面字节的前两位都设为10，这n个字节的其余空位填充该字符unicode码，高位用0补足。
>
>   ```js
>   // unicode
>    I 00000000 01001001
>    知 01110111 11100101
>   ```
>   ```js
>   // utf-8
>    I 01001001
>    知 11100111 10011111 10100101
>   ```



## utf-16

> 编码规则
>
> 可变长
>
>  0–65535 范围内的字符编码成 2 个字节，超出部分编码为4个字节
>
> 英文、汉字：2个字节，表情：4个字节
