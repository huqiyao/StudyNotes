# 几种路径

| 路径          | 含义                                                         |
| ------------- | ------------------------------------------------------------ |
| __dirname     | 获得当前执行文件所在目录的完整目录名（该dirname写在哪，就代表这个文件的绝对路径） |
| __filename    | 获得当前执行文件的带有完整绝对路径的文件名（该dirname写在哪，就代表这个文件的绝对路径） |
| process.cwd() | 获得当前执行node命令时所在文件夹的目录名                     |
| ./            |                                                              |
| ../           |                                                              |
|               |                                                              |

- 想检验```./```  ```../``` 代表的绝对路径是什么时，```console.log(path.resolve('./'));```
